# Gradle

Install Gradle [바로가기](https://gradle.org/install/)

##### Version

```
$ gradle -v
```

##### Add Wrapper

```
$ gradle wrapper
```

##### Change Wrapper Version

```
$ gradlew wrapper --gradle-version 8.3
```

##### Gradlew Tasks List

```
$ gradlew tasks
```

##### Gradlew Refresh

```
$ gradlew --refresh-dependencies
```
